I make functional products which are inspired by my communities and try to make their lives easier, or just a little more fun. Click on any of the projects below to find our more about them!

  • Stu the Stuorg Bot

    • Stu is a Telegram bot designed and implemented by me in Python for Yale-NUS' Student Affairs Office. It's used by students to answer FAQs, make classroom booking requests, and check for upcoming events. Try out using Stu here.

    • This app was built as part of Yale's CPSC419 Full Stack Web Programming module, improving the lackluster Yale Shuttle app to be more user-friendly.
